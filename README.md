![Java](https://img.shields.io/badge/Java-11-orange)
![Maven](https://img.shields.io/badge/Maven-3.6+-red)
![License](https://img.shields.io/badge/License-Educational-blue)

Учебный проект по разработке быстрой сортировки с использованием паттернов проектирования.

## 📋 Описание проекта

Приложение для сортировки объектов класса `Car` с использованием алгоритма быстрой сортировки (QuickSort). Проект демонстрирует применение паттернов проектирования Стратегия и Builder, работу с коллекциями, файлами и многопоточностью.

## ✨ Основные возможности

- **Сортировка объектов**: Реализация быстрой сортировки с использованием паттерна Стратегия
- **Управление данными**:
  - Ввод данных вручную
  - Генерация случайных данных
  - Загрузка данных из файла
  - Валидация данных при вводе
- **Работа с коллекциями**:
  - Создание и заполнение коллекций с использованием стримов
  - Реализация кастомных коллекций
- **Работа с файлами**:
  - Запись отсортированных данных в файл
  - Добавление данных в существующий файл
- **Многопоточность**: Подсчет количества вхождений элемента в коллекцию

## 🚀 Установка и запуск

### Клонирование репозитория

```bash

```


## 🚗 Класс Car

Класс `Car` представляет автомобиль со следующими полями:
- **Мощность** (int) - мощность автомобиля в лошадиных силах
- **Модель** (String) - название модели автомобиля
- **Год производства** (int) - год выпуска автомобиля

**Реализует:**
- Паттерн Builder для удобного создания объектов
- Интерфейс Comparable для сортировки по всем полям

## 🔢 Алгоритмы сортировки

- **QuickSort** - основной алгоритм сортировки
- Реализация вручную без использования готовых библиотечных решений
- Сложность: O(n log n) в среднем случае

## 🎨 Паттерны проектирования

- **Стратегия** (Strategy) - для выбора алгоритма сортировки
- **Строитель** (Builder) - для создания объектов класса Car
